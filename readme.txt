To Run:

--First time:

chmod +x run.sh
./run.sh

--Other times:

./run.sh

--Obs1: 

Put your inputs inside "instances" folder, it will automatically read all of them

--Obs2:

If you want to test your inputs to see if something is missing in the pattern, run ./run.sh testAll
For this, you must also have your inputs inside "instances" folder
